# EMR On EKS Management Tools Desktop

Desktop application for EMR on EKS operations.

Target Users: Data Engineer

Features:

- Multiple Profiles
- Submit Spark Job
- View and Monitor Job Status
- View CloudWatch Logs
- Manage Job Templates
- S3 File edit

## Tech Stack

Frontend:

- React 19
- Typescript
- Tailwind
- Shadcn UI

Backend:

- Tauri v2
- Rust
